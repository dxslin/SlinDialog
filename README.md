### 使用方法：

1. 使用前请在你的build.gradle添加 `maven { url 'https://jitpack.io' }`，比如
```
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```

2. 然后根据需要添加依赖，比如
```
implementation 'com.github.dxslin:SlinDialog:v0.1'
```
